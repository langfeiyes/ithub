# 三、super关键字

super 含义：

this ：表示当前对象

super ：当前对象的父类对象

什么时候使用super: 需要在子类中调用父类的成员时，一定使用super

  • 在子类方法中，调用父类被覆盖的方法，此时必须使用super

  • 在子类构造器中，调用父类构造器，此时必须使用super语句

    语法：
super();
super(参数1，参数2...)

# 四、static 修饰符

# 4.1 static 修饰符

static 修饰符表示静态，可以修饰成员变量构成静态成员变量，修饰成员方法构成静态成员方法。

静态成员变量和静态成员方法，称为类的静态成员，都归类所有

# 4.2 静态变量

在类中，用static修饰符修饰的成员变量称为静态成员变量，归类所有，语法 :

[修饰符] static 数据类型 变量名;
1

静态变量的访问语法：

类名.静态变量名		 // 推荐写法
对象名.静态变量名		// 底层用 类名.静态变量名 方式访问

2

# 4.3 静态方法

static 也可以修饰成员方法称为静态成员方法，归类所有。

[修饰符] static 返回值类型 方法名(形参列表){
    方法体;
}

此时，观察发现类中的成员 按照是否被static修饰来分：分为静态成员和非静态成员 静态成员（静态变量、静态方法等），归类所有，推荐使用类名来调用 非静态成员，归对象所有，必须通过实例（对象）来调用

记住结论 ( 十分重要 )：

  • 静态成员：使用static修饰的字段和方法： 属于类 直接使用类名调用即可

  • 非静态成员：没用static修饰的字段和方法： 属于对象 必须先创建对象，再调用

注意：static方法不能使用super和this

因为static是类级别的，super和this是对象级别的
存在类的时候不一定存在对象，也就说使用类名调用static方法时，此时可能是没有对象的。

# 4.4 final修饰符（掌握）

子类可以继承父类的实现细节，也可以通过方法覆盖的形式修改功能实现细节。那么怎么来限制某个类不能有子类，不能覆盖方法？——final修饰符

# 4.5 代码块 (了解)

什么是代码块：直接使用{ }括起来的一段代码区域。

根据位置的不同，存在三种形式：

  • 局部代码块： 直接定义在方法内部的代码块，一般的，不会直接使用局部代码块的，结合if、while、for等关键字使用，表示一块代码区域。代码块中的变量属于局部变量，只在自己所在区域{ }内有效。
public class CodeBlockDemo {
	public static void main(String[] args) {
		System.out.println("begin...");
		{
			// 直接使用代码块，一般不用
			int age = 17;
		}
        System.out.println(age);	//此行报错,超出age作用范围,就不能访问到了
        
		int[] arr = { 1,3,5 };
        for( int i = 0;i < arr.length;i++ ) {
            int item = arr[i];
            System.out.println(item);
        }
		System.out.println("end...");
	}
}

# 4.6. 内部类

# 4.6.1 内部类分类 ( 了解 )

什么是内部类，把一个类定义在另一个类的内部，把里面的类称之为内部类，把外面的类称之为外部类。（能认识内部类即可）

内部类可以看作和成员变量、方法一样，是外部类的成员，而且成员可以有static修饰。

  • 静态内部类：使用static修饰的内部类，那么访问内部类直接使用外部类名来访问

  • 实例(成员)内部类：没有使用static修饰的内部类，访问内部类使用外部类的对象来访问

  • 局部(方法)内部类：定义在方法中的内部类，一般不用

  • 匿名内部类：特殊的局部内部类，适合于仅使用一次使用的类

对于每个内部类来说，Java编译器会生成独立.class文件。

  • 静态和实例内部类：外部类名$内部类名字

  • 局部内部类：外部类名$数字内部类名称

  • 匿名内部类：外部类名$数字

只需要记住：如果一个类字节码名字中含有$，那么它存在内部类

# 4.6.2 匿名内部类 (重点掌握)

  • final修饰的方法：最终的方法, 该方法不能被子类覆盖

  • final修饰的变量：表示常量，该变量只能赋值一次，不能再重新赋值。

  • 基本数据类型：表示的值不能改变

  • 引用数据类型：所引用的地址值不能改变

public class Student {

    public static double getAvg(Student[] students) {
        // step 1: 定义一个变量sum，存储求和
        double sum = 0;
        // step 2: 遍历数组，取出每个学生对象，获取学生的成绩，累加到sum中
        for(Student stu:students) {
            sum += stu.getScore();
        }
        // step 3：求平均值
