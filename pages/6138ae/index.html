# 岁月多蹉跎，吾辈需努力

# 岁月多蹉跎，吾辈需努力

这篇来谈谈学Java的前提 (以下观点仅代表个人浅见，如果有不同意见，我当没看见。)

  • 亲人支持

    行军打仗，最忌讳的是祸起萧墙。家人，亲人就是你大后方，如果他们都不支持你学，那就放弃吧。就别听那些培训机构推出的感人案例：家人不支持，就瞒着家人，破釜沉舟，然后一举成功。家人不和，成功有啥意义，如果真心想学，好好跟家人沟通。

  • 学历

    学习Java有学历要求么？有，也没有。

    先说有，一个好的学历能证明什么？能证明你曾经很优秀，比如，你本科清华毕业，可以证明高中你很优秀，你硕士也是清华毕业，证明你大学很优秀，如果你是博士，那你一直都很优秀。一个优秀的人，都有一套科学的学习方法，那学Java至少事半功倍。再者，学成之后去就业，有了学历加成，已经拥有了先手优势，别人拿青砖敲门，你却用上金砖。

    再说没有，只要你智商正常，还真可以学。Java入门简单精通难，学到小成(应付普通项目开发)，不需要太高的学历。只要你学习得法，肯坚持，肯付出，石头也能开出小花啦。

    最后，我并不是啥唯学历论的人，我见过小学学历能学成的，也见过硕士学历从入门到放弃的，高学历在IT这行并不是必须品，但绝对是不可或缺的，我始终相信，学历决定你上限。

  • 学习态度

    书说，学习态度一般是指学生对学习及其学习情境所表现出来的一种比较稳定的心理倾向。这说法完全赞同，学习Java在你没有找到学习快感之前，学习Java就是一个苦逼事情。在你没悟道之前，对java学习持有啥态度就决定你是否有缘分走这条路啦。

    有句话怎么说来的，思路决定出路，态度决定高度，格局决定结局。思路，态度，格局，3个哲学题；思路，格局，就不说啦，没资格讨论，就说态度。因为一些原因，我接触过很多学习Java的朋友，同等条件下有人短时间内完成蜕变，晋升IT新丁；也有人一致在门口徘徊，迟迟不得法，如此差异原因很多，笼统的说，因人而异；本质原因：成事决于态度。

    另外，态度不是句口号，而是一日复一日，一年复一年的坚持，一个未来可能是一场空的坚持。

  • 兴趣

    我接触过很多学java的朋友，跟他们聊过去学Java的想法，收集了好多：1>喜欢IT这行，借java入行； 2>IT这行工资高，我想努力一下; 3>亲戚/朋友干这行，推荐我走这样； 4>技多不压身，多学一门傍身; 5>之前工作不好，转行; 6>老师/家长让我来学的.....经过多年的跟踪观察，悟道较为容易是：喜欢这行的跟工作不好转行。

    喜欢这样行，好理解，兴趣是学习最好的驱动力，也是最好的老师。当然也不是说有兴趣就能学好Java，还需要其他综合条件，比如学习态度，是否能坚持等。说到这，就得讨论一个问题，如何算是感兴趣？一个很简单判断技巧，去做一件你想去做的事情，并坚持一星期，然后问自己开心不，还想不想再做一个星期，如想，那就感兴趣，如果不想，那就放弃吧。

    工作不好转行，这个更好理解，穷是学习原动力，不接受反驳。

  • 持之以恒

    决定来学java之前，好好问自己，对待一个未知的学习，你想报以啥态度？是持之以恒，习惯成自然呢，还是三天打渔两天晒网？如果你对学习Java没有敬畏心，还是放弃吧，它没这个福分。 做人，做事讲究"贵有恒，何须三更起五更眠；最无益，只怕一日曝十日寒"，学Java亦如此。

    当然，坚持就是胜利，这废话差不多，初学者最需要的是方法论，不是毒鸡汤。但转头一想，寻找科学的，持续的方法论也是你学习一部分，坚持并科学去探索就能取得胜利，说的也没错。

  • 觉悟

    学习是孤独，没有人义务去帮你；也没有坚持就一定能胜利这说法；也没有什么现成方法论，科学的捷径；你更不是天之骄子，更做不到一学就会，一听就懂....

    所以，承认自己是普通人，脚踏实地去学，小心翼翼经营你与Java的感情，这才是一个Java学者的觉悟。
