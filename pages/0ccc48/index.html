# 2.1 数据类型（重点）

生活中，数据都是有类型这个概念的，比如张三18岁，18这个数字就是整型的，买了2.5斤菜，2.5就是小数类型的，在Java中每一个数据也有数据类型。

在计算机中，人为地把8个二进制位称为一个字节(byte)

8大基本数据数据类型所占内存大小 和可存储的数据范围：

No. 数据类型 占位(字节) 数据范围
1 byte 1 [ -128 , 127 ]
2 short 2 [ -32768 , 32767 ]
3 int 4 [ -2^31^ , 2^31^ - 1 ] ≈ [-21亿，21亿]
4 long 8 [ -2^63^ , 2^63^-1 ]
6 float 4 [-3.4x10^38^ , 3.4x10^38^]
7 double 8 [-1.7x10^308^ , 1.7x10^308^]
5 char 2 [0 , 2^16^-1]
8 boolean 1个二进制位 true,false
提醒：开发者需要明确记住每个具体数据类型所占字节数（内存大小）。

# 2.2.1 变量概述（了解）

变量指在程序运行过程中，值可以发生变化的量。

变量是内存中一块区域，可以往该区域存储数据，修改里面的数据，也可以获取里面的数据。

变量定义的语法：

数据类型 变量名 = 初始值;
1

变量的特点：

  • 占据着内存中的某一块存储区域
  • 该区域有自己的名称（变量名）和大小（数据类型）
  • 可以被重复使用
  • 该区域的数据可以在同一类型范围内不断变化

# 三、基本数据类型转换

问题：三个大小不同容器，能相互把盛装的水倒给对方吗？

在8大基本数据类型中，boolean不属于数值类型 ( 主要用于逻辑判断 )，所以不参与转换。

转换规则如下图：

按照转换方式，有两种：

  • 自动类型转换：范围小的数据类型直接转换成范围大的数据类型 (小 => 大)。
  • 强制类型转换：范围大的数据类型强制转换成范围小的数据类型 (大 => 小)。

注意：

  • 一般的，byte、short、char三种类型之间一般不参与转换。
  • float类型表示的数据范围比long大
科普了解:
float占4个字节为什么比long占8个字节大？---> 因为底层的实现方式不同
float类型的32位并不是简单直接表示大小，而是按照一定标准分配的。
第1位，符号位，即S
接下来8位，指数域，即E。
剩下23位，小数域，即M，取值范围为[1 ,2 ) 或[0 , 1)
然后按照公式：V = (-1) ^ s * M * 2 ^ E  
也就是说小数float在内存中的32位不是简单地转换为十进制，而是通过公式来计算而来，通过这个公式，虽然只有4个字节，但小数最大值要比长整型的范围要大。

# 3.1 自动类型转换（掌握）

自动类型转换，也称为"隐式类型转换"，就是把范围小的数据类型直接转换成范围大的数据类型。

转换规则：byte、short、char—>int—>long—>float—>double

语法格式

范围大的数据类型 变量 = 范围小的数据类型常量值/变量;
1

语法举例：

public class TypeConvertDemo {
    public static void main(String[] args){
        int intNum1 = 10;			// 没有发生类型转换
        long longNum2 = intNum1;	// 发生自动类型转换

        long longNum3 = 30L;
        float floatNum4 = longNum3;

        // 思考: 以下代码是否正确？
        float floatNum5 = 3.14f;
        double doubleNum6 = floatNum5;
    }
}

# 3.2 自动类型提升（掌握）

当一个算术表达式中，包含多个基本数据类型的常量或变量（boolean除外）时，整个算术表达式的结果类型将在出现自动提升， 其规则是：

  • 所有的byte、short、char类型被自动提升到int类型，再参与运算

  • 整个表达式的最终结果类型，被提升到表达式中数据范围最大的那个类型

public class TypeConvertDemo2 {
    public static void main(String[] args){
        int result = 'a' + 1; 	// 'a'转成int后是97
        System.out.println(result);

        double d1 = 123 + 1.1F + 3.14 + 99L;
    }
}

# 3.3 强制类型转换（掌握）

强制类型转换，也称为“显式类型转换”，就是把范围大的数据类型强制转换成范围小的数据类型。

语法格式：

范围小的数据类型  变量 = (范围小的数据类型)范围大的数据类型值;
1

注意：一般情况下不建议使用强转，因为强转有可能损失精度

public class TypeConvertDemo3 {
    public static void main(String[] args){
        // 强制类型转换
        float floatNum1 = 3.14f;
        int intNum = (int)floatNum1;
        System.out.println(intNum);

        // 应用:根据消费金额计算vip积分(规则：一块钱积2分)
        float price = 998.88f;
        int vipScore = (int)price * 2;
        System.out.println(vipScore);

        // 思考题:给定一个字符类型变量c，里面存了一个字符 请将它转成字母表中的下一个字符
        char c = 'a';
        char c2 = (char)(c + 1);
        System.out.println(c2);
    }
}

# 4.1.2 自增和自减（掌握）

自增：++，递增操作符，使变量值增加1，有前置和后置之分，只能操作变量。

自减：-- ，递减操作符，使变量值减去1，有前置和后置之分，只能操作变量。

自增和自减具体操作是一样的，仅仅是一个是加1，一个是减1而已，现在单讲++。

代码 i++和 ++i，结果都是 i 变量的值加1。

唯一的区别是：

  • 前置（++i）: 表示对result加1之后的结果进行运算

  • 后置（i++）: 表示对result变量加1之前的值（原始值）进行运算。

public class ArithOperatorDemo2 {
    public static void main(String[] args) {
        // i++: 遇到i++，i先参与运算，运算完成后i自加1 => 速记:先使用后增加
        // 情况1:
        /*int i = 10;
        i++;
        System.out.println(i);*/

        // 情况2:了解
        /*int i = 10;
        int j = i++;
        System.out.println(i);
        System.out.println(j);*/

        // ++i: 遇到++i, i先自加1，然后再参与运算
        // 情况1:
        /*int i = 10;
        ++i;
        System.out.println(i);*/

        // 情况2:了解
        int i = 10;
        int j = ++i;
        System.out.println(i);
        System.out.println(j);
    }
}

所以，如果仅仅执行简单的递增操作（只写i++ 或 ++i），那么选用任意一个都可以。

总结结论：

  • 前置++ ：先增加后使用

  • 后置++ ：先使用后增加

# 4.4 三元运算符（掌握）

三元运算符，表示有三个元素参与的运算符，所以又称为三目运算符，其语义表示：如果什么情况就做什么，否则做什么。

语法格式：

数据类型 变量 = boolean表达式 ?  结果A : 结果B；
1

如果boolean表达式结果：

  • 为true，则三元运算符的结果是结果A；

  • 为false，则三元运算符的结果是结果B；

注：三元运算符必须定义变量接收运算的结果，否则报错。

另外，三元运算符结果的类型由结果A和结果B来决定的，结果A和结果B的类型是相同的。

需求1：判断一个数99是不是偶数

public class TernaryOperatorDemo1 {
	public static void main(String[] args) {
		int a = 99;
		String result = a % 2 == 0 ? "偶数" : "奇数";
		System.out.println(result);
	}
}

# 4.5 逻辑运算符（掌握）

逻辑运算符用于连接两个boolean表达式，结果也是boolean类型的。

语法格式为：

boolean  result = boolean表达式A  逻辑运算符  boolean表达式B;
1

逻辑运算符常用的有三个：

  • && 与运算，理解为且，并且
    • 运算规则：只要两边的表达式有一个为false，结果就为false（有false则false）
  • || 或运算，理解为或，或者
    • 运算规则：只要两边的表达式有一个为true，结果就为true （有true则true）
  • 非：取反操作
    • 运算规则：!true则false , !false则true

public class LogicalOperatorDemo1 {

		System.out.println(false || true); // true
		
        //相反操作
		System.out.println(!true);  // false
        System.out.println(!false); // true
	}
}

# 4.5.2 短路运算（掌握）

&&：如果&&左边表达式为真，&&右边表达式参与运算；如果&&左边表达式为假，&&右边表达式不参与运算，故称短路与。

|| : 如果||左边表达式为true，|| 右边表达式不参与运算

public class LogicOperatorDemo2 {
    public static void main(String[] args) {
        // 逻辑运算符
        // 短路与 &&
        int m = 10;
        int n = 20;
        // boolean r1 = false && true;
        boolean r1 = (m > n) && (++m > 1);
        System.out.println("r1 = " + r1);
        System.out.println("m = " + m);

		// 短路或 ||
        boolean r2 = (m > n) || (++m < 1);
        System.out.println("r2 = " + r2);
    }
}
